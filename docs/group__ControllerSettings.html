◆ TC42SetShortCircuitDetectionSettings()

DLL_RESULT TC42SetShortCircuitDetectionSettings ( unsigned short  maxPWM,
unsigned short  voltageDifference 
)

Sets short circuit detection parameters. This function can improve the control behavior in the event of a short circuit.

Parameters
[in]maxPWMReduces controller output to X% of actual value when detecting a short circuit [0...4096]
0: 0%
4096: 100%
[in]voltageDifferenceMinimal current difference between two cycles (50us) to detect a short circuit[0...4000].
Attention
voltageDifference is not a physical value - so translate to correct physical value
Note
Setting short circuit detection parameters on a TopCon Master will automatically transfers them to all connected Slaves.
Setting short circuit detection parameters on a TopCon Slave will have no effect.
Short circuit detection is supported from firmware V4.22.30 or later and is only effective in Q1 operating mode.
See also
Chapter Translate between standardised values and physical values
Returns
DLL_SUCCESS for success or DLL_FAIL or other value if an error occurs
